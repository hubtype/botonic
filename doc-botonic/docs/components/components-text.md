> ## Purpose

Text is the simplest and easiest way to talk to the user.

> ## Code

The "Hello World!" output would be defined as below:

```javascript
<Text>Hello World!</Text>
```

In order to create a line break, you must use `{'\\n'}`.

```javascript
<Text>
  First line {'\\n'}
  Second line
</Text>
```
